# Changelog

All notable changes to Exile Farm Ledger are documented here.
The format is based on Keep a Changelog and this project follows Semantic Versioning.

## [0.1.4] - 2026-06-27
### Added
- Pause and resume an active farming session; paused time is excluded from duration and per-hour metrics.
- The update banner now shows patch notes for the incoming release.

## [0.1.3] - 2026-06-27
### Added
- Enter currency values in Divine Orbs (1 decimal) via a unit toggle on the Prices screen.

## [0.1.2]
- Baseline release (see GitHub release notes).
